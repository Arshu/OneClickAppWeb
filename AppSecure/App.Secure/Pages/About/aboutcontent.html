Welcome to the Secure Demo Application running on Secure App Runtime
Secure App Runtime is a Common Abstraction Layer implemented using Arshu Web Components for packaging applications using a meta-data service oriented architecture (both frontend and backend)
Portable/Installable AppWeb Architecture
Using Multi-Platform AppWeb Runtime
The apps runtime frontend is an embedded webview/openglview and the backend is an embedded extendable multi-platform Json Service

The packagable apps frontend can be written in Html, Css, and Javascript and the backend can be written in C#, Lua, Objective-C, Java
