App Web Runtimes provide common abstraction across multiple platforms implemented natively in each platform to enable development of portable apps.
The AppWeb Runtime provides a common abstraction across multiple platforms enabling portablity from one environment to another and can be writen in the language of the platform or using a cross platform toolkit. The API Service of the AppWeb Runtime is written in C#, F#, Lua, ObjC and Java.
Many example MIT Licensed App Web Runtime source having different common abstractions are published in Github
