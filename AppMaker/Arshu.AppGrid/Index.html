Arshu Application Runtime is a Common Abstraction Layer implemented using Arshu Web Components in Native Mobile Platform for packaging/developing applications using a meta-data service oriented architecture (both frontend and backend)

This example Sample App demonstrates the various embedded web services API to access device functionality.
