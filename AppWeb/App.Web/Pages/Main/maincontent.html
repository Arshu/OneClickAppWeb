Welcome to the Sample Web App running on Arshu Application Runtime
Arshu Application Runtime is a Common Abstraction Layer implemented using Arshu Web Components in Native Mobile Platform for packaging/developing applications using a meta-data service oriented architecture (both frontend and backend)
The apps runtime frontend is an embedded webview/openglview and the backend is an embedded extendable multi-platform Json Service

The packagable apps frontend can be written in Html, Css, and Javascript and the backend can be written in C#, Lua, Objective-C, Java

This example MIT Licensed appweb runtime source is published in


Github (github.com/Arshu/OneClickAppWeb)
