Arshu AppWeb is a Offline WebApp Viewer for viewing of static/ dynamic(ajax) web apps without internet

  1. Main features are
  2. Offline viewing of static/ dynamic(ajax) zipped web apps
  3. No hosting/domain required for sharing static/dynamic web apps
  4. Privately share using email/ btsync/ dropbox etc
  1. The main use case are
  2. Developers building mobile webapps can share their web apps without hosting it in a server
  3. Designers/Developers building dynamic mockups/prototypes can share without hosting it in a server
  4. Developers/Testers testing mobile hybrid apps can test on actual device without creating a mobile hybrid app
  1. Instructions
  2. Zip your mobile web apps with index.html in the root folder and rename the file extension as pap (Portable App)
  3. Copy it to the mobile phone using Btsync, Dropbox etc
  4. Import the (Portable App) pap file to AppWeb Viewer
  1. Manage AppWeb(s)
  2. Reimport your imported web apps
    (Inbox Action)
    using the round left button
  3. Remove your imported web apps
    (Remove Action)
    using the round left button
  4. Reimport
    the embedded Web App using the round left button
  5. Refresh the Web App using the round right button
